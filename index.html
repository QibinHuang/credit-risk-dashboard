Credit Risk Analysis and Fraud Detection Dashboard

This interactive dashboard project demonstrates how financial institutions can use data-driven analytics to improve credit risk assessment and fraud detection. Built with Power BI, SQL, and Python, the project simulates over 2 million customer credit records to provide actionable insights.

Project Overview

Using machine learning models (Random Forest, XGBoost), the system predicts credit risk scores, detects fraudulent activities, and reduces false positives. The dashboard visualizes trends across credit scores, loan amounts, fraud patterns, and regional risk.

The solution also includes model performance evaluation using ROC and precision-recall metrics, offering transparency into fraud prediction capabilities.
