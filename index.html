Project Summary

This Power BI dashboard simulates over 2 million customer credit records to demonstrate how financial institutions can apply data-driven solutions for credit risk assessment and fraud detection. Built using SQL, Python, and Power BI, the project combines machine learning models (Random Forest, XGBoost) with interactive data visualizations.
